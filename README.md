# vue-travel

> 一个 vue-cli搭建的vue全家桶小型项目，适合入门vue

### 启动项目

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```



### 项目目录

```
├─build    
├─config    
├─src
│  │  App.vue
│  │  main.js
│  ├─assets		// 静态资源       
│  ├─common		// 公共组件    
│  ├─pages		// 页面组件   
│  ├─router		// 路由
│  ├─store		// vuex
│  └─utils		// 常用工具函数
│          
└─static
│  .babelrc
│  .editorconfig
│  .gitignore
│  .postcssrc.js
│  index.html
│  list.txt
│  package.json
│  README.md
```
